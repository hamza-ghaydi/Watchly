# Notification Sound

To enable notification sounds, add a file named `notification.mp3` to this directory.

## Quick Setup Options:

### Option 1: Use a free sound from the web
Download a notification sound from sites like:
- https://notificationsounds.com/
- https://mixkit.co/free-sound-effects/notification/
- https://freesound.org/

### Option 2: Create your own
Use online tools like:
- https://www.beepbox.co/
- https://sfxr.me/

### Option 3: Use a system sound
Copy a notification sound from your system:
- Windows: `C:\Windows\Media\`
- macOS: `/System/Library/Sounds/`
- Linux: `/usr/share/sounds/`

## File Requirements:
- Format: MP3
- Name: `notification.mp3`
- Duration: 1-3 seconds recommended
- Size: Keep under 100KB for best performance

Once added, the notification system will automatically play this sound when new notifications arrive.
